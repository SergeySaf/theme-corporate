Animations

Trigger animations to elements when they become visible.

Fade in scale 

<div class="bg-faded p-4 rounded" data-animate="fadeInScale">
  Fade in scale
</div>

Data options

data-animate="EFFECT" to the element you want to animate.

data-animate-delay="TIME" Time in seconds or milliseconds to add a delay.

data-animate-duration="TIME" Time in seconds or milliseconds to set the duration of the effect (default is 1 second).

data-animate-infinite="true" To run the animation in a loop.


Example:
Run again
Hello world, how are you!? 

<span data-animate="fadeIn">Hello</span>
<span data-animate="fadeIn" data-animate-delay=".4">World,</span>
<span data-animate="fadeIn" data-animate-delay=".9">how</span>
<span data-animate="fadeIn" data-animate-delay="1.4">are</span>
<span data-animate="fadeIn" data-animate-delay="1.8">you!?</span>
